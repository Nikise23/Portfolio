# Portfolio - Nicolás Fernández

Portfolio profesional de desarrollador Full Stack con foco en Backend.

## Stack Tecnológico

- React 18
- Vite
- TailwindCSS

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```

## Estructura del Proyecto

```
src/

│   ├── Hero.jsx

│   ├── Projects.jsx
│   ├── ProjectCard.jsx
│   ├── Skills.jsx
│   ├── Contact.jsx
│   └── Footer.jsx
├── App.jsx
├── main.jsx
└── index.css
```
